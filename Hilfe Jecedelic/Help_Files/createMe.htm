createMe();


Verwendung

createMe("Callback", "Clientname", "Produktname");


Parameter

Callback:
Die Funktion, in die der Server mit den Parametern antworten soll.

Clientname:
Der Clientname ist der Nickname des Clients der sich am Server anmelden möchte. Dieser Name wird über ein Input-Textfeld oder z.B. eine Clientname-Generator Funktion definiert.

Produktname:
Der einmalige Produktname der Anwendung. Er dient der sinnvollen Unterscheidung auf dem Server, da mehrere verschiedene Anwendungen gleichzeitig mit dem Server kommunizieren können.
Beispiele: "Tobis Chat", "Panzerkommandant 4", etc.


Rückgaben in Callback-Funktion

  1. "true" oder "false"
    Gibt zurück, ob die Registrierung erfolgreich war.
     
  2. clientID
    Die eindeutige clientID des Clients, welche einmalig auf dem Server ist.
     
  3. clientName
    Der Name des Clients auf dem Server.

Beschreibung

Erstellt den aktuellen Benutzer der Anwendung als Client auf dem Server. Obwohl die Verbindung zum Server schon vorher erfolgte, wird der Benutzer erst jetzt wirklich registriert.
Wenn die Anfrage erfolgreich war, d.h. der Clientname noch nicht vergeben wurde oder die maximale Anzahl an Clients nicht überschritten wurde, gibt der Server als ersten Parameter ein "true" (String) zurück, ausserdem noch 2 weitere Parameter.

Falls die Anfrage nicht erfolgreich war, wird kein weiterer Parameter übergeben.


Tipp:

Sinnvoll ist es hier, die Callback Funktion nach dem ersten Parameter zu steuern, sprich anhand einer if-Abfrage nach dem "true" oder "false", wie auch im folgenden Beispiel.
