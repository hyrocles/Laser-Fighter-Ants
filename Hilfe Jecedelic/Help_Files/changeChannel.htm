changeChannel();


Verwendung

changeChannel(Callback, ChannelName, Passwort)


Parameter

Callback:
Die Funktion, in die der Server mit den Parametern antworten soll.

Channelname:
Der Name des Channels in den gewechselt werden soll.

Passwort:
Für Passwortgeschützte Channels.


Rückgaben in Callback-Funktion

  1. "true" oder "false"
    Gibt zurück, ob der Channel erfolgreich gewechselt werden konnte.
     
  2. Channelname
    Wenn "true" zurückgegeben wird, sendet der Server noch den Channelnamen mit.
     

Beschreibung

Erstellt einen Channel auf dem Server. Wenn alles glatt läuft, d.h. der Channelname noch nicht vergeben wurde oder die maximale Anzahl an Channels nicht überschritten wurde, gibt der Server als ersten Parameter ein "true" (String) zurück. Falls die Anfrage nicht erfolgreich war, wird "false" (String) zurück gegeben.

Die Parameter Channelname, Modus und Passwort sind in ihrer Verwendung optional können zur weiteren bearbeitung der Anwendung dienen.


Anmerkung:

Wenn der Channelnam.
