jecConnect();


Verwendung

jecConnect(Callback, Host, Port);


Parameter

Callback:
Die Funktion die nach erfolgreichem/misslungenem Verbindungsaufbau aufgerufen werden soll.

Host:
Die IP oder der Hostname des zu Jecedelic Servers.

Port:
Die Portnummer auf die der Jecedelic Server listened (meistens 6937).


Rückgaben in Callback-Funktion

  1. true oder false
    Verbindung aufgebaut oder misslungen

Beschreibung

Baut eine XMLSocket Verbindung zum Jecedelic Server auf, welcher unter angegebener IP oder Hostname erreichbar ist. Der angegebene Port ist der Netzwerk-Kanal, welcher zur Kommunikation mit dem Server dient. Der Jecedelic Server kommuniziert standardmäßig über den Port 6937.


Anmerkung:

Sollte es nicht möglich sein, einen aktiven Server über den gewählten Port zu kontaktieren, vergewissern Sie sich, dass der gewählte Port auch vom Server gelistened wird und keine Firewall diese Kommunikation unterbindet.
