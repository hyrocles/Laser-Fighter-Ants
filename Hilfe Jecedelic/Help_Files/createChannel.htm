createChannel();


Verwendung

createChannel("Callback", "Channelname", "Modus", "Passwort");


Parameter

Callback:
Die Funktion, in die der Server mit den Parametern antworten soll. Meist kann man hiermit die Hauptanwendung starten oder in eine Art Wartezimmer wechseln, um auf andere Mitspieler zu warten.

Channelname:
Der Name des Channels der neu angelegt wird.

Modus:
Eine Eigenschaft des Channels. Hier kann zum Beispiel ein Spielmodus wie "Team Match" oder "Turbo Mode" definiert werden.

Passwort:
Für Passwortgeschützte Channels.


Rückgaben in Callback-Funktion

  1. "true" oder "false"
    Gibt zurück, ob der Channel erfolgreich angelegt wurde.
     
  2. Channelname
    Name des angelegten Channels
     
  3. Modus
    Modus des angelegten Channels
     
  4. Passwort
    Passwort des angelegten Channels

Beschreibung

Erstellt einen Channel auf dem Server. Wenn die alles glatt läuft, d.h. der Channelname noch nicht vergeben wurde oder die maximale Anzahl an Channels nicht überschritten wurde, gibt der Server als ersten Parameter ein "true" (String) zurück. Falls die Anfrage nicht erfolgreich war, wird "false" zurück gegeben.

Die Parameter Channelname, Modus und Passwort sind in ihrer Verwendung optional können zur weiteren bearbeitung der Anwendung dienen.


Tipp:

Sinnvoll ist es hier, die Callback Funktion nach dem ersten Parameter zu steuern, sprich anhand einer if-Abfrage nach dem "true" oder "false", wie auch im folgenden Beispiel.

function ausgabe(erfolg, Channelname, Modus){
    if(erfolg == "true"){
        gotoAndStop(Modus);
        _root.channelname.text = Channelname;

    }
}
